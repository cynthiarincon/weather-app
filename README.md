A simple weather application that allows users to search for current weather conditions in any location.

## Live Demo

**Deployed on Render:** https://weather-app-zlzq.onrender.com

## Features

- Search weather by city name
- Real-time weather data from WeatherStack API
- Location geocoding using Mapbox API
- Responsive design
- Clean, modern UI

## Technologies Used

- **Node.js** - Backend runtime
- **Express** - Web framework
- **Handlebars (HBS)** - Templating engine
- **Request** - HTTP client for API calls
- **CSS** - Styling

## APIs Used

- [WeatherStack](https://weatherstack.com/) - Weather data
- [Mapbox](https://www.mapbox.com/) - Geocoding

## Local Setup

1. Clone the repository:
```bash
git clone https://github.com/cynthiarincon/weather-app.git
cd weather-app
```

2. Install dependencies:
```bash
npm install
```

3. Run the application:
```bash
node src/app.js
```

4. Open your browser and visit:
```
http://localhost:3000
```
